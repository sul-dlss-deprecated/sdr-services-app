Overview

This API provides a RESTful interface to the Archive Catalog for the Stanford Digital Repository. This is SDR metadata, primarily data for tracking data replication.

Content is referenced by digital repository unique identifiers (DRUIDs). A DRUID is a unique identifier, such as 'jq937jp0017' or 'druid:jq937jp0017'. The DRUID regex pattern, using posix bracket notation, is:

[[:lower:]]{2}[[:digit:]]{3}[[:lower:]]{2}[[:digit:]]{4}
